What makes an RFQ ready for supplier review?

A botanical extract RFQ becomes easier to answer when the buying team separates technical requirements from commercial assumptions. The first message does not need to be perfect, but it should include enough context for the supplier to identify the correct product path rather than guessing from a common ingredient name.

  • Ingredient name, botanical name if known, plant part, and target extract form.
  • Preferred active marker, extract ratio, test method, or standard grade.
  • Application format such as capsule, tablet, gummy, powder blend, beverage, food, or personal care.
  • Estimated sample, pilot, first order, and repeat-order quantities.
  • Document needs such as COA, TDS, and available testing file questions.

Standard extract, bulk supply, or custom specification?

Not every project should follow the same sourcing route. A standard extract request may move quickly through specification and document review, while a custom botanical extract may need more technical discussion before sample timing and MOQ can be confirmed. Buyers can save time by explaining whether they need a standard grade, a wholesale bulk quote, or a made-to-order specification.
